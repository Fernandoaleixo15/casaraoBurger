<h1>🍔 Casarão Burger</h1>
<p>O <span> Casarão Burger</span> é um sistema de cardápio digital interativo que permite aos usuários visualizarem produtos, adicionarem itens ao carrinho e finalizarem pedidos via WhatsApp. Focado em uma experiência de usuário (UX) fluida e design responsivo.</p>
<h2>🚀 Demonstração</h2>
<a>https://casarao-burger-9v8pew0fp-fernandos-projects-28d50bb4.vercel.app/</a>
<h2>🛠️ Tecnologias Utilizadas</h2>
<p>O projeto foi construído utilizando o trio fundamental do desenvolvimento web:</p>
<ul>
  <li>HTML5: Estruturação semântica do conteúdo.</li>
   <li>CSS3: Estilização, layout (Flexbox/Grid) e animações.</li>
   <li>JavaScript (ES6+): Lógica do carrinho, manipulação do DOM e integração com a API do WhatsApp.</li>
</ul>
<h2>✨ Funcionalidades</h2>
<ol>
  <li>Cardápio Dinâmico: Visualização de hambúrgueres e bebidas.</li>
  <li>Carrinho de Compras: Adicionar, remover e atualizar quantidades de itens.</li>
  <li>Validação de Horário: Sistema que verifica se a hamburgueria está aberta antes de permitir o pedido.</li>
  <li>Integração com WhatsApp: Envio automático do resumo do pedido para o número do estabelecimento.</li>
  <li>Design Responsivo: Adaptado para celulares, tablets e desktops.</li>
</ol>
